# Conception et développement de RAM Dumper, outil d’extraction contrôlée du contenu de la mémoire vive sous NetBSD

# Objectifs:
1. Comprendre l’organisation mémoire sous NetBSD
2. Manipuler des interfaces bas niveau de gestion de la mémoire
3. Mettre en œuvre un outil de forensic ou debugging de la mémoire
4. Garantir l’intégrité du système

# Fonctionnalités attendues
1. Lecture de la mémoire
2. Extraction de blocs mémoire définis 
4. Enregistrement dans un fichier dump 
5. Vérification des permissions et des accès
